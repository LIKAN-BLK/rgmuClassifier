Описание метода.
Для классификации из размеченной  и разделенной на целевые и нецелевые эпохи ээг извлечены два набора признаков - вейвлетные и амплитудные.

Метод из извлечения см. в описании к коду извлчения признаков. 
Вейвлетные фичи упакованы в два 4-х мерных (масштаб х временной отсчет х канал Х попытка) массива, содержащих 
целевые и нецелевые эпохи. 
Всего 51 масштаб, 251 временной отсчет и 23 каналов. Так же вместе с вейвлетными признаками идет маска размерностью масштаб х время. Маска определяет релевантные признаки.

Так же в процессе работы формирется вторая маска, отвчающая за децимацию вейвлетных фич. Маски применяются совместно. 
Порядок децимации для каждого масштаба различный. Для каждого
масштаба оставляется количество признаков, равное соответствующей частоте, умноженной для длинну интервала и умноженной на определенный целочисленный
коэффициент. 
В данных экспериментах коэффициент был равен 4. Переход от масштаба вейвлета к частоте fq = -0.5*scale +30.5.
Далее отсекались 3 последних канала, соответсвующих окулограмме и признаки вытягивались в матрицу, 
где по строкам были попытки, а по столбцам - признаки.

Амплитудные фичи использовались напрямую.
Для подсчета эффекивности алгоритма использовался 4-х проходной скользящий контроль (СК). 
Внутри СК выборки амплитудных и вейвлетных фич делились на
тестовую и тренировочную. Для вейвлетных признаков считалась r2 метрика между вектором призннаков и вектором меток. 
Далее отсекались 70% признаков, чья
 r2 метрика была наименьшей. Потом по отдельности к наборам амплитудных и вейвлетных фич применялся PCA и в итоговый вектор состоял из 80-ти главных компонент
вейвлетных признаков
 и 80-ти главных компонент амплитудных признаков. Таким образом каждый объект выборки имел 120 признаков. Далее тренировочная выборка
предавалась на вход классификатору shrinkLDA.
 Далее тренировочная выборка умножалась на матрицу классификатора и по результату подбирался порог классификации, который
 обеспечит специфичность 0.99.

Так же были поставлены два эксперимента, при которых для классификации использовались отдельно амплитудные и вейвлетные признаки с теми же параметрами 
перцентили и главных компонент.
